import file to our code.

Modules: 
    - core (dependency)
    - npm packages
    - local modules

- core (dependency) importing ex:
	const fs = require('fs');
	const path = require('path');
	const http = require('http');

- npm packages importing ex:
	install using npm >npm install express
	import it in code :
		const express = require('express');

- local modules:
	the module opSum.js :
		const sum = (a, b) => {
    			return a + b;
		}
		module.exports = {
    			"summation" : sum
		}
	
	import in a file
		const sum01 = require("./opSum");
		console.log(sum01.summation(11111, 33));


*** we can use import for : core, npm, local modules also
	import fs from 'fs';
	import express from 'express';
	import myUtils from './opSum.js';
